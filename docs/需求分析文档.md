## 功能说明

### 指令分析模块

#### 数据词典

合法指令格式

#### 功能

1. 分析指令

   输入一行指令，判断是否合法，调用其它模块执行，输出。

### 账户管理模块

#### 数据词典

账号，权限，密码。

#### 功能

1. 注册（0）

   输入：账号，密码。

   输出：注册成功/失败。

2. 登录（0）

   输入：账号，密码。

   输出：登录成功/失败。

3. 注销（1）

   输入：无。

   输出：注销成功/失败。

4. 修改密码（1）

   输入：账号，原密码，新密码。

   输出：修改成功/失败。

5. 创建用户（2）

   输入：账号，密码。

   输出：创建成功/失败。

6. 删除用户（3）

   输入：账号。

   输出：删除成功/失败。

### 图书管理模块

#### 数据词典

ISBN 号（20 个字符，每本正版书都有惟一的标识）、书名（60 个字符、作者（60 个字符）、关键字（可以有多个，总共不超过 60 个字符）、库存量（整型）、单价（非负实数）

#### 功能

1. 查询图书（1）

   输入： ISBN 号、作者名字、关键字、书名中的任意组合。

   输出：所有满足要求的图书信息。

2. 购买图书（1）

   输入：购买书籍的 ISBN 号和数量。

   输出：购买成功/失败。

3. 录入图书（2）

   输入：书籍完整信息

   输出：录入成功/失败

4. 修改图书（2）

   输入：ISBN 号和欲修改的新信息

   输出：修改成功/失败

5. 进货（2）

   输入：ISBN 号，数量，进货价格。

   输出：进货成功/失败。

### 日志模块

#### 数据词典

财务日志，员工日志。

#### 功能

1. 查询采购信息（3）

   输入：查询时间段。

   输出：采购图书相关信息。

2. 查询销售情况（3）

   输入：查询时间段。

   输出：销售图书相关信息。

3. 查询盈利信息（3）

   输入：查询时间段。

   输出：收入、支出、利润。

4. 查询员工工作报告（3）

   输入：员工账号，时间段。

   输出：员工操作信息。

5. 查询系统工作日志（3）

   输入：查询时间段

   输出：全部操作信息。
